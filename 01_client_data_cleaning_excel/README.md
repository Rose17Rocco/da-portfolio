# Client Data Cleaning and Enrichment
**Goal**
Improve data quality in a B2B client list for accurate reporting, segmentation, and sales analysis.
---
## Tools Used
- Microsoft Excel (Formulas, Conditional Formatting, Data Validation)
  -  Power Query (for merging, cleaning, and deduplication)
    ---
  ## Key Stages
  1. **Data Cleaning**
     - Removed duplicates and standardized company name formats.
     - Split combined address fields into City/State/Zip.
     - Corrected inconsistent phone and email formats.
    
  2.**Data Enrichment**
  - Added missing information such as industry classification.
  - Created calculated fields.
  - Used VLOOKUP to cross-reference external data.
 
  3.**Quality Checks**
  - Used conditional formatting to flag missing or invalid entries.
  - Built pivot tables to verify total record counts before/after cleaning.
 
  ---
  ## Results
  - Reduced duplicate client entries.
  - Standardized 100% of address and contact formats.
  - Prepared dataset for CRM integration and SQL import.
